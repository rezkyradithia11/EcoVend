## Deskripsi Karya

Website ini membantu pengguna mengelola sampah dengan cara menyetor sampah ke bank sampah. Setiap aktivitas ramah lingkungan akan memberikan poin yang nantinya bisa ditukar dengan produk ecofriendly di marketplace. Selain itu, pengguna juga dapat mengikuti misi harian berbasis konsep 3R (Reduce, Reuse, Recycle) untuk mendapatkan poin dan membangun kebiasaan.
